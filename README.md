*A comprehensive collection of optimized DSA solutions demonstrating clean code, algorithmic proficiency, and problem-solving skills.*

## 🎯 Purpose

This repository serves as a personal library of Data Structures and Algorithms implemented from scratch in **Java**. It demonstrates my ability to solve complex technical problems, write clean and maintainable code, and analyze time and space complexities. 

It includes foundational data structure implementations alongside optimized solutions to common algorithmic challenges from platforms like LeetCode, HackerRank, and GeeksforGeeks.

---

## 🚀 Key Highlights for Recruiters

* **Algorithmic Proficiency:** Covers a wide spectrum of topics ranging from basic array manipulation to advanced Dynamic Programming and Graph algorithms.
* **Clean Code Practices:** Solutions adhere to standard Java naming conventions, modular design, and Object-Oriented Programming (OOP) principles.
* **Performance Focused:** Solutions are built with efficiency in mind. Code files include documented **Time** and **Space Complexity** (Big O notation) in the comments.
* **Edge Case Handling:** Algorithms are rigorously tested against edge cases, null checks, and boundary conditions.

## 📂 Repository Structure

The project is organized logically by topic to allow for easy navigation and code review.

| Category | Description | Key Concepts Explored |
| :--- | :--- | :--- |
| **Arrays & Strings** | Array manipulation and string processing | Two Pointers, Sliding Window, Prefix Sum |
| **Linked Lists** | Custom node implementations and traversals | Reversal, Cycle Detection, Fast/Slow Pointers |
| **Trees & Graphs** | Hierarchical data and network structures | BFS, DFS, Binary Search Trees, Shortest Path |
| **Dynamic Programming** | Optimization and memoization techniques | Knapsack, Fibonacci sequence, Matrix traversals |
| **Sorting & Searching** | Foundational algorithms for data retrieval | Merge Sort, Quick Sort, Binary Search |
| **Stacks & Queues** | LIFO and FIFO data structures | Monotonic Stacks, Priority Queues, Heaps |
